Overview

Service Name:

Memory Status (Fujitsu)

Service ID:

24199

Service Type:

SNMP

Supported Devices / OS:

Fujitsu workstations, laptops or servers running the Fujitsu ServerView agent.

Maximum Service Instances / Device:

100

Service Description:

Monitors the Status, Size, Form, Type, and whether or not the RAM is approved for Use in a Fujitsu device.

MIB Module(s):

SC2.MIB

Notes:

Configuration

In order for this service to work, the Fujitsu device must have the Fujitsu ServerView Agent installed, and have SNMP enabled and configured.

 

The Windows version of the Fujitsu ServerView Agent can be downloaded here.

 

For a list of the Installed Memory on a Fujitsu server, query it with the 1.3.6.1.4.1.231.2.10.2.2.10.6.5.1.3 SNMP OID.

Miscellaneous

The Status metric is mapped as follows:

 

Normal: 3, 8, 9, 10, 11

Warning: 7

Failed: 1, 2, 4, 5, 6

 

1: Unknown

2: Not Present

3: OK

4: Disabled

5: Error

6: Failed

7: Prefailure Predicted

8: Hot Spare

9: Mirror

10: RAID

11: Hidden

Errors

Misconfigured on the dashboard probably means that an invalid remote IP address has been specified, that the SNMP community string of the device isn’t correct, or that the Fujitsu ServerView Agent hasn’t been installed.
