Notes

Configuration

Enter the name of the Volume to be monitored. For a list of Hard Drives, query the QNAP NAS with the 1.3.6.1.4.1.24681.1.2.17.1.2 SNMP OID.

Errors

Misconfigured on the dashboard probably means that the SNMP community string of the device isn’t correct, or that an incorrect Volume Name has been specified.

Comments

Due to how QNAP implemented their MIB, the Status metric in this service is a string value. As such, this service will never report a Failed status, and should only be used for informational purposes.
